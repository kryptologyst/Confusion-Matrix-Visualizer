# Confusion Matrix Visualizer

**Confusion matrix** generation with raw and normalized views.

## Overview

- Train a classifier and generate confusion matrix
- Raw counts and row-normalized views
- Classification report with per-class metrics
- **Streamlit dashboard** with Plotly heatmap

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
# CLI: python -m src.main visualize --dataset wine
pytest tests/ -v
```

## Docker

```bash
docker compose up --build
```
